# **GS-Odyssey-5.41**
*A GameServer for Fortnite Season 5 (5.41)*

---

## **Features**

1. Looting 1:1
2. Build/Repair/Edit 1:1
3. Pickup 1:1
4. Respawn works perfectly
5. The percentage of chests/ammunition boxes works
6. Llama spawn works
7. BGA consumables spawn
8. The vending machine works
